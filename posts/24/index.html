CSS 꿀팁

안녕하세요 임기입니다 :D

CSS 꿀팁을 몇 가지 적어보고자 글을 씁니다.


저는 프로젝트를 시작하게 되면 가장 먼저 추가하는 스타일이 있는데요

바로 box-sizing 스타일입니다.

* {
  box-sizing: border-box;
}


박스사이징을 설명하기 전에 일반적인 상황을 먼저 보여드릴게요.

박스사이징을 설정하지 않은 상태에서 width : 100px인 버튼을 만들고 싶다면 아래와 같이 설정하게 됩니다.

button {
  width: 100px;
  padding: 8px;
  border: 1px solid black;
}


그런데 이렇게 되어있다면 실제로 button의 width는 100 + 8*2 + 1*2 = 118px이 됩니다.

저희가 원하는 크기는 100px인데 훨씬 커지므로 레이아웃이 깨져서 width를 82px로 줄여줘야하는 불편함이 있죠 :(


이런 불편함을 해결할 수 있는게 box-sizing 스타일입니다.

박스사이징을 설정하게 되면 border, padding, content(width, height) 요소가 content 사이즈를 넘어가지 않게 됩니다.

즉, width: 100px을 설정하면 자동으로 border와 padding을 감안해서 width가 82px인 것 처럼 보이게 되는 것이죠.

박스사이징은 width: 100% 처럼 퍼센티지 요소를 적용할 경우도 아주 잘 작동하게 됩니다.


단, 조심할 점이 하나 있는데요. margin은 width 밖에 적용되므로 margin을 적용해야하는 경우는 부모 엘리먼트의 전체 너비를 넘지 않도록 사이즈 조정을 해주어야 레이아웃이 깨지지 않습니다.


저는 이럴 때 CSS 함수인 calc()를 주로 사용합니다.

예를 들어서 menu와 content를 좌우로 배치하는 경우를 생각해봅시다.

<!-- example.html -->
<body>
  <div id="menu"/>
  <div id="content"/>
</body>



/* example.css */
#menu{
  width: 300px;
}

#content{
  width: 100%;
}


위와 같이 스타일을 설정하면 menu와 content의 너비의 합이 100% + 300px이 되어서 레이아웃이 깨지게 됩니다.

그래서 이 두개를 같이 정렬하기 위해서는 width를 계산해서 입력해줘야합니다.

아래와 같이 말이죠.

/* example.css */
#menu{
  width: 300px;
}

#content{
  width: calc(100% - 300px);
}


주의 해야할 점은 calc() 안에 사칙연산 좌우로 띄어쓰기를 해주어야 잘 작동합니다. 띄어쓰기 없이 붙여쓰게 되면 CSS가 동작하질 않아요.


세번 째 꿀팁은 display: flex 입니다 :D

flex는 요소들을 좌우 또는 상하 배치할 때 요긴하게 사용됩니다.


아래 예시를 볼까요?

<!-- example.html -->
<div class="flexDiv">
  <button>1번</button>
  <button>2번</button>
  <button>3번</button>
  <button>4번</button>
</div>



/* flex.css */
.flexDiv{
	background: lightblue;
	width: 100%;
	height: 100vh;

	display: flex;
	justify-content: space-between;
	align-items: flex-start;
}
button{
	background: green;
	color: white;
}


위 스타일을 적용하면 아래 그림 처럼 버튼이 배치되게 됩니다.

.flexDiv 안에 버튼들이 나란히 위치해 있죠? 이걸 가능하게 해주는 속성이 justify-content입니다.

justify-content에는 몇 가지 값들이 있는데 이걸 알아두셔야 상황에 맞는 스타일을 사용하실 수 있습니다.

justify-content는 기본 축(default: row) 방향으로 요소들을 배치합니다. 기본 축이 가로이므로 한 줄에 여러개 요소들이 배치되는 모습이죠.

  • space-between : 좌우 끝에 요소를 하나 씩 배치하고 남은 요소들을 가운데에 균일하게 배치합니다. 보통 페이지의 헤더 부분에 많이 사용하는 방식이죠. 위에 있는 그림처럼 배치됩니다.
  • space-around : 모든 요소를 균일하게 배치합니다. 아래와 같은 모양이죠

  • flex-start : 축의 시작(왼쪽)에 요소들을 배치합니다. 딱히 flex를 쓸 필요가 없겠죠?

  • flex-end : 축의 끝(오른쪽)에 요소들을 배치합니다.
  • center : 축의 가운데에 요소들을 배치합니다.


다음으로 알아볼 속성은 align-items입니다.

align-items는 justify-content와 반대 축을 기본 축으로 가집니다. justify-content가 가로 축이었다면 align-items는 세로 축이 되겠죠?
