CSS 애니메이션에 대해 알아보자

안녕하세요 임기입니다 :D


CSS 애니메이션에 대해서 알아볼건데요.

간단한 사용 예시는 아래와 같습니다.

/* animation.css */
#refresh {
  width: 48px;
  height: 48px;
}

.refresh {
	animation: refreshing 1s linear infinite;
}

@keyframes refreshing  {
	0% {
		transform: rotate(0deg);
	}
	100% {
		transform: rotate(360deg);
	}
}


사용방법은 정말 간단합니다.

@keyframes 뒤에 애니메이션 이름을 적어줍니다. 여기서는 refreshing입니다.

그리고 프레임 마다 어떤 동작을 할지 적어주시면 됩니다.

프레임은 0% ~ 100% 사이의 퍼센티지를 작성해주시면 프레임 간에 연속된 애니메이션이 동작됩니다.


이제 애니메이션을 언제 시작하고 종료할지 고민하셔야합니다.

예시는 refresh이니 새로고침 버튼을 만들어서 버튼을 누르면 애니메이션이 동작하고, 버튼을 다시 누르면 애니메이션이 종료되게 해봅시다. 저는 주로 querySelector를 이용해서 classList를 추가하고 제거하는 방법을 사용합니다.

<!-- example.html -->
<body>
  <img src="./refresh.png" id="refresh"/>
  <div>
    <button onClick="refreshing()">새로고침</button>
  </div>

  <script src="./example.js"></script>
</body>



// example.js
let refreshToggle = false
const refresh = document.querySelector("#refresh")
function refreshing() {
  if (refresh) {
      if (!refreshToggle) { // toggle이 false이면 클래스 추가
        refresh.classList.add("refresh")
        refreshToggle = true
      } else { // toggle이 true이면 클래스 제거
        refresh.classList.remove("refresh")
        refreshToggle = false
      }
  }
}


새로고침 버튼을 클릭하면 refresh 클래스가 추가되면서 애니메이션이 동작하고, 버튼을 한 번 더 클릭하면 클래스가 삭제되면서 애니메이션이 사라지는 예시입니다.


이제 animation 속성에 대해서 좀더 알아볼건데요.

animation에는 animation-name, delay, direction, duration, fill-mode, iteration-count 속성 등이 있습니다.

  • name: @keyframes에서 만든 애니메이션 이름
  • delay: 애니메이션이 시작하기 전 지연 시간. default) 0s
  • direction: 애니메이션의 동작 방향. 정방향은 normal, 역방향은 reverse, alternate는 매 사이클마다 방향을 전환.
  • duration: 애니메이션이 한번 반복되는 시간. ex) 1s
  • fill-mode: 애니메이션이 끝날 때의 스타일 지속여부. none, 스타일 지속하고 싶으면 normal. default) none
  • iteration-count: 애니메이션 반복 회수. ex) 1, 1.5, infinite


저는 보통 한 줄에 표현하는 걸 선호합니다.

animation: refreshing 1s linear infinite;

이것은 refreshing 애니메이션을 1초 동안 linear하게 무한히 반복하겠다. 라는 뜻이랍니다.


여기서 linear는 애니메이션의 속도를 조절하는 속성인데요. linear는 일정하게, ease는 부드럽게 라고 생각하시면 됩니다.

일정하게 회전해야 하는 경우는 보통 linear를 사용하구요. 위치나 크기가 변경되는건 ease를 사용하면 부드럽게 표현된답니다.
