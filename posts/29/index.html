[React 정리] 1장) 초기 환경 세팅하기, create-react-app으로 앱 만들기

안녕하세요 임기입니다 :D

React로 프로젝트를 하면서 정리를 할 필요가 있다고 생각이 들어서 글을 써봅니다.


React 앱을 만들기 위한 강의를 시작해보겠습니다~

저는 React를 처음 공부할 때 create-react-app을 이용했는데요. 이후에도 직접적인 바벨 설정을 건드릴 필요가 없을 경우 계속해서 CRA로 앱을 만들고 있습니다. CRA는 노드가 설치되어있으면 바로 실행할 수 있어요.


노드가 설치되어있지 않다면 node.js를 설치해줍니다. nodejs 홈페이지로 접속해서 안정적인 버전으로 설치하세요. 어렵지 않게 설치하실 수 있을거에요 :D


다음은 cmd나 git-bash를 실행합니다. 저는 리눅스 명령어들이 더 친숙해서 bash를 더 자주 사용합니다. 배쉬를 실행하셨다면 앱을 만들어 볼까요?

npx create-react-app [프로젝트명]을 입력해보세요.


좀 기다리시면 진행상황이 보이며 몇 분 내로 다운로드가 끝나게 될거에요.


자! 다운로드가 끝났으면 디렉토리로 들어가 볼까요?

저는 프로젝트명을 example로 만들었기 때문에 cd example 을 입력합니다.
다음으로 code . 을 입력합니다.

code . 은 현재 디렉토리를 visual studio code로 실행해줍니다. 참고로 explorer . 도 있는데요 이건 탐색기로 현재 디렉토리를 열어줍니다.


짜잔, vsc가 실행되셨나요?


vsc는 Microsoft사에서 제작한 오픈소스 에디터랍니다. 전 세계에서 가장 많이 사용하는 에디터죠. vsc를 사용하고 있지 않으시다면 이참에 사용해보는건 어떨까요? ㅎㅎ


이제 본론으로 넘어와서 vsc왼쪽 상단의 문서처럼 생긴 버튼을 눌러서 Explorer를 열고 public 폴더의 index.html을 클릭해 봅시다.

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8" />
    <link rel="icon" href="%PUBLIC_URL%/favicon.ico" />
    <meta name="viewport" content="width=device-width, initial-scale=1" />
    <meta name="theme-color" content="#000000" />
    <meta
      name="description"
      content="Web site created using create-react-app"
    />
    <link rel="apple-touch-icon" href="%PUBLIC_URL%/logo192.png" />
    <link rel="manifest" href="%PUBLIC_URL%/manifest.json" />
    <title>React App</title>
  </head>
  <body>
    <noscript>You need to enable JavaScript to run this app.</noscript>
    <div id="root"></div>
  </body>
</html>


간단한 html파일이 열리는데요. 이 index.html이 리액트 앱에서 실행되는 단 하나의 페이지입니다. 리액트앱을 Single Page Application이라고 하죠? 여기서 싱글페이지가 이 index 파일을 의미한답니다. 하나의 페이지에서 js로 렌더링하여 화면을 바꾸는 것이죠. 그래서 전통적인 웹페이지와 다른 특수한 장치들이 필요하답니다. 그건 조금 후에 알아볼게요.


index파일을 간단하게 설명해볼게요

<head> 태그는 리액트앱의 기본설정들이에요.

  • meta charset : 페이지 인코딩 설정
  • link rel="icon" : 즐겨찾기 아이콘 이미지 설정
  • meta name="viewport" : 디바이스에 따른 화면크기 설정, 스케일 설정
  • meta name="theme-color" : 테마컬러 설정. 여기서는 검은색으로 설정했네요
  • meta name="description" : 페이지 내용 소개(검색 사이트 노출시킬 때 필요)
  • link rel="apple-touch-icon" : 애플터치 아이콘 이미지 설정
  • link rel="manifest" : 모바일 앱 설정. 모바일 앱을 만들 때 각종 설정파일을 manifest에 작성해주는데 아이콘 이미지, 앱 이름 등을 설정해주는 부분이랍니다.
  • title : 페이지 제목 설정


<body> 태그에는 <noscript>와 <div id="root"></div>밖에 없습니다.

<body>
    <noscript>You need to enable JavaScript to run this app.</noscript>
    <div id="root"></div>
  </body>

<noscript>는 스크립트를 실행할 수 없는 환경에서 나타나는 문구입니다.

여기서 중요한건 바로 <div>인데요. 이 root div에 리액트 앱을 렌더링 한다는 걸 꼭 기억하세요 !!


글이 꽤 길어졌네요. ㅎㅎ 다음 강의에서 계속 할게요~

다음 강의는 리액트앱의 구조를 살표보고 앱을 실행해봅시다!
