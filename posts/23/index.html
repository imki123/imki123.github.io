리액트 contextAPI useContext() 적용기

안녕하세요?

블로그를 제작하면서 Redux 등의 상태관리 라이브러리를 사용하지 않다가 오늘 리액트에서 제공하는 contextAPI useContext()를 적용해봤습니다.


리액트는 컴포넌트에서 상태값들을 관리 할 수 있는데, Top-down 방식으로 상위 컴포넌트의 상태를 하위 컴포넌트에게 전달해줄 수 있습니다. 문제는 상태값을 전달하기 위해 수 많은 컴포넌트를 거쳐야하는 props drilling이 매우 빈번하게 발생할 수 있습니다.


이를 보완해주는 것이 Context API였고 useContext() Hook으로 발전하면서 함수형 컴포넌트에서도 손쉽게 사용할 수 있게 되었습니다.


사용법은 꽤나 간단합니다.

루트 컴포넌트에 store를 만들어서 전역상태로 관리하고 싶은 상태값들을 객체 형태로 담습니다.

// App.js
const [login, setLogin] =  useState(false)
const [user, setUser] =  useState({ username: 'imki123' })
const sayHello = () => { console.log('Hello!') }
//전역상태값들을 store에 집어 넣습니다. 네이밍은 편한대로 사용하시면 됩니다 :D
const store = {
  login, setLogin, user, setUser, sayHello,
}

그 다음 App 컴포넌트 위쪽에 AppContext = React.createContext()를 생성하고 아래쪽에서 export { AppContext } 합니다. 다음으로는 <AppContext.Provider>로 App을 감싸주고 value에 store를 담아 하위 컴포넌트들에게 전달해줍니다.

// App.js
import React, { useState } from 'react'

const AppContext = React.createContext()

function App() {
  const [login, setLogin] =  useState(false)
  // ...
  const store = {
    login, setLogin, user, setUser, sayHello,
  }
  // ...

  return (
    <AppContext.Provider value={store}>
      <div id="app">
        <Second>
          <Third>
            <Fourth/>
          </Third>
        <Second/>
      </div>
    </AppContext.Provider>
}

export { AppContext }
export default App


그리고 AppContext의 하위 컴포넌트에 가서 AppContext를 import하고 store = React.useContext(Appcontext)를 불러오면 props drilling 없이 store 객체를 사용할 수 있습니다.

// Fourth.js
import React from 'react'
import { AppContext } from './App'

function Fourth() {
  const store = React.useContext(AppContext)

  const logout = e => {
    store.setLogin(false)
    store.setUser(false)
  }
  return(
    {login && 
    <div>
      <div>{store.user.username}님 환영합니다 :D</div>
      <button onClick={logout}>로그아웃</button>
    </div>}
  )
}

export default Fourth


어때요 쉽죠? :D

세션 정보같이 여러 곳에서 사용해야 하는 전역 상태는 store로 관리하고 간단한 상태들은 props로 전달하면 훨씬 깔끔한 앱을 만드는데 도움이 될거에요~
